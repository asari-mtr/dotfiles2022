# dotfiles2022

## Install Brew
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## Install from brew

```
brew bundle install --global --verbose
```

## Install tpm for tmux
```
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```
